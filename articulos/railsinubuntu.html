Instalación de Ruby on Rails en Ubuntu 12.04 con RVM.

(11 Julio 2013)

  1. Instalar Ruby con RVM.

    2. Aseguramos que todos los paquetes están actualizados para ello ejecutamos el siguiente comando:

    $ sudo apt-get update

    Una vez realizado la actualización procedemos a la instalación de RVM. Si no tienes instalado curl(herramienta para transferir archivos) procedemos a la instalación.

    $ sudo apt-get install curl

    Una vez instalado curl, instalamos RVM.

    $ \curl -L https://get.rvm.io | bash -s stable

    Una vez instalado cerramos la sesión en el terminal y creamos una nueva y procedemos a cargar de nuevo RVM.

    $ source ~/.rvm/scripts/rvm

    $ echo "source \$HOME/.rvm/scripts/rvm" >> ~/.bashrc

    RVM tiene algunas dependencias que necesitan ser instaladas, para verlas necesitamos ejecutar el siguiente comando:

    $ rvm requirements

    A continuación nos aparecerá las siguientes dependencias:

    apt-get --no-install-recommends install build-essential openssl libreadline6 libreadline6-dev curl git-core zlib1g zlib1g-dev libssl-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt-dev autoconf libc6-dev libgdbm-dev ncurses-dev automake libtool bison subversion pkg-config libffi-dev

    Por lo que ejecutaremos la instrucción.

    $ sudo apt-get --no-install-recommends install build-essential openssl libreadline6 libreadline6-dev curl git-core zlib1g zlib1g-dev libssl-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt-dev autoconf libc6-dev libgdbm-dev ncurses-dev automake libtool bison subversion pkg-config libffi-dev

  2. Instalar Ruby.

    3. Instalamos ruby con RVM.

    $ rvm install 1.9.3

    Ahora le decimos al sistema que utilice ruby 1.9.3 por defecto.

    $ rvm use 1.9.3 --default

  3. Instalar RubyGems.

    4. Para instalar RubyGems utilizamos también Rvm.

    $ rvm rubygems currents

  4. Instalar Rails.

    5. $ gem install rails

    Este proceso suele tardar un poco pero una vez finalizado ya tienes instalado Ruby on Rails en tu máquina.
