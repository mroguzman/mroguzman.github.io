EnNómina

Junio (2013)

Es una herramienta web para el cálculo de nóminas online. Una empresa puede registrar a todos sus empleados para posteriormente proceder al cálculo de las nóminas, poder exportarlas a PDF así como su almacenamiento en la nube. La aplicación esta desarrollada en Ruby mediante el framework de desarrollo web Ruby on Rails y PostgreSQL como base de datos.
